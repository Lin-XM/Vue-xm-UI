## 开始使用

1. 添加 CSS 样式  
使用之前，请在 css 中开启 `border-box`。  
```css
*，*::before,*::after{
    box-sizing: border-box;
}
```
还需要设置一些默认颜色变量：
```css
:root {
    --button-height: 32px;
    --font-size: 14px;
    --button-active-bg: #eee;
    --button-bg: white;
    --border-radius: 4px;
    --color: #999;
    --border-color: #999;
    --border-color-hover: #666;
}
```
